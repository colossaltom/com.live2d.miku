# Hatsune Miku in an Illustrator panel

Adds live2D Hatsune Miku to her own Illustrator/Photoshop panel!!1

...thats it. She follows your mouse around watching what you do. Have fun.

## Install
Install dist/zxp/com.live2d.miku.zip using a zxp installer like aeplugins

https://aescripts.com/learn/post/zxp-installer


## How to edit
Build environment BOLT using react. Documentation on how to use it can be found on its github:

https://github.com/hyperbrew/bolt-cep

Hatsune Miku was added using a wonderfully simple plugin documented here:

https://github.com/Konata09/Live2dOnWeb


### Live2D
settings file located in:

src/js/main/jsx/waifu-tips.js

model files located in:
